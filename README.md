# 📊 Task 2 – Customer Retention & Churn Analysis Dashboard

## 🚀 Internship Project | Future Intern

This project was completed as **Task 2 during my internship with Future Intern**.  
The goal of this project was to build an **Interactive Customer Retention & Churn Analysis Dashboard** to analyze telecom customer data and extract insights that help businesses understand customer behavior and improve retention strategies.

---

## 🔍 Key Insights from the Dashboard

- Customer churn analysis by **Contract Type**
- Churn distribution across **Internet Service Types**
- Relationship between **Monthly Charges and Customer Churn**
- Impact of different **Payment Methods on Churn**
- **Tenure vs Churn** analysis to identify early customer drop-off
- Effect of services like **Tech Support and Online Security on retention**

---

## 📊 Dashboard Features

- Interactive filters for deeper analysis
- KPI cards for:
  - Total Customers
  - Churn Rate
  - Retention Rate
  - Average Monthly Charges
  - Average Tenure
- Visual analysis of customer behavior patterns
- Business insights for improving customer retention
